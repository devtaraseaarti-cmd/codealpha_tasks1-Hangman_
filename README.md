# Hangman Game (Python)

## Description
This is a simple text-based Hangman game written in Python.  
The player guesses the word one letter at a time with limited attempts.

## Features
- Predefined list of words
- Maximum 6 wrong attempts
- Console-based game
- Beginner friendly

## Technologies Used
- Python
- random module

## How to Run
1. Open terminal in the project folder
2. Run the command:
   python hangman.py
